## Getting Started

- [The Nim Programming Language](https://nim-lang.org/) – official website & documentation
- [Nim Tutorial (Part 1)](https://nim-lang.org/docs/tut1.html) – the best place to start
- [Nim Basics](https://narimiran.github.io/nim-basics/) – beginner-friendly guide
- [Nim by Example](https://nim-by-example.github.io) – learn by reading real code
- [Nim Playground](https://play.nim-lang.org/) – try Nim instantly in the browser

## Books

- [Nim in Action](https://book.picheta.me/) by Dominik Picheta – the classic, build real projects (chat app, web app, parser)
- [Computer Programming with Nim](https://www.nimprogrammingbook.com/book/nimprogramming_colorful.pdf) – free gentle introduction (PDF)
- [Mastering Nim](https://www.amazon.com/Mastering-Nim-complete-programming-language/dp/B0B4R7B9YX) – complete deep dive
- [Nim for Beginners](https://www.amazon.com/Nim-Beginners-Mastering-Programming-Languages-ebook/dp/B0FRWGCZKT) – modern entry point

## Interactive Tutorials & Exercises

- [Exercism Nim Track](https://exercism.org/tracks/nim) – 70+ exercises with mentoring (highly recommended)
- [Nim by Example](https://nim-by-example.github.io)
- [Nim on Rosetta Code](https://rosettacode.org/wiki/Category:Nim) – hundreds of solved problems

## Projects & Applications

### CLI Tools & Apps
- [nimmm](https://github.com/joachimschmidt557/nimmm) – fast terminal file manager for Linux
- [niffler](https://github.com/gokr/niffler) – command-line AI assistant written in Nim

### Web & Full-stack
- [Jester](https://github.com/dom96/jester) – Sinatra-like web framework (super easy DSL)
- [Prologue](https://github.com/planety/prologue) – full-featured full-stack web framework
- [Happyx](https://github.com/HapticX/happyx) – macro-powered async full-stack framework
- [Karax](https://github.com/karaxnim/karax) – build single-page applications entirely in Nim

### Games & Graphics
- [nico](https://github.com/ftsf/nico) – Pico-8 style game framework (fantastic for retro-style games)
- [godot-nim](https://github.com/pragmagic/godot-nim) – first-class Nim bindings for Godot Engine
- [frag](https://github.com/Tail-Wag-Games/frag) – modern cross-platform 2D/3D game engine

### Systems & Embedded
- [natu](https://github.com/exelotl/natu) – complete toolkit for writing Game Boy Advance games in Nim

### Other Cool Projects
- [nimforum](https://github.com/nim-lang/nimforum) – lightweight Discourse-style forum (powers forum.nim-lang.org)

## Tools & Utilities

- [choosenim](https://github.com/nim-lang/choosenim) – effortless Nim version manager
- [nimble](https://github.com/nim-lang/nimble) – Nim package manager (bundled with the compiler)
- [nimlangserver](https://github.com/nim-lang/langserver) – official language server (VS Code, Neovim, etc.)
